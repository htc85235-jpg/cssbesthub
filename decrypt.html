How does this work? / How to use on Android

How decryption works:

  1. The .enc file from your email contains 3 parts: a 16-byte salt, a 12-byte IV, and the AES-GCM ciphertext.
  2. Your password is stretched into a 256-bit key using PBKDF2 (100,000 rounds, SHA-256).
  3. That key + IV decrypt the ciphertext. If the password is wrong, AES-GCM's auth tag fails and you'll see an error.

On Android (Chrome):

  1. Save the .enc attachment from Gmail to your device (Downloads folder).
  2. Open this page in Chrome. Bookmark it for next time.
  3. Tap the file picker above → "Files" → choose the .enc file.
  4. Password slowdumbo is already filled in. Tap Decrypt & Play.
  5. Audio plays inline. Tap Download to save the decrypted audio (.webm / .mp4 / .ogg).

Works fully offline: After the first load, you can use this page without internet. To install as an app: Chrome menu → "Add to Home screen".

Privacy: Everything happens locally in your browser. No file or password is ever uploaded anywhere.
